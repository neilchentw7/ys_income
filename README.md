# 應收帳款視覺化分析

此專案提供一個以 **Streamlit** 製作的簡易網頁應用程式，協助使用者分析 Excel 檔案中的應收帳款資料。

## 安裝依賴

```bash
pip install -r requirements.txt
```

## 執行方式

於專案目錄下執行：

```bash
streamlit run app.py
```

## Excel 檔案格式

上傳的工作簿需至少包含以下三個工作表：

- `應收帳款表`
- `銷售日報輸入`
- `銷售月報`

程式會自動計算出貨量、應收帳款總額與稅後平均單價，並產生對應的表格及長條圖。

## 主要功能

- 顯示本月出貨數量和應收款總金額
- 計算各客戶稅後平均單價
- 依應收帳款金額分類並繪製圖表
